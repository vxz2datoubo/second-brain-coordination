## 核心算法

### SM-2 间隔重复算法
```
新知识: 1天 → 6天 → 复习次数 × 2.5
评分<3: 重置为1天
```

### 遗忘曲线 (艾宾浩斯)
```
刚学: 100% 记住
1天后: 33.7% 记住
1周后: 25.4% 记住
1月后: 21.1% 记住
```

### 跨域联想领域
1. **交易** ↔ 心理 (情绪管理)
2. **交易** ↔ 数学 (概率/期望)
3. **交易** ↔ 物理 (动量/惯性)
4. **交易** ↔ 博弈 (零和博弈)
5. **心理** ↔ 生物 (适应性)
6. **数学** ↔ 物理 (定律描述)
7. **博弈** ↔ 系统 (复杂性)

## 使用示例

### Python API

```python
from core.cognitive_memory_system import get_cognitive_memory_system

cms = get_cognitive_memory_system()

# 创建原子化笔记
note_id, warnings = cms.create_atomic_note(
    title="追高的风险",
    content="追高买入往往导致当日被套，应等待回调 #教训 #交易",
    tags=["教训", "交易"],
    para_category="A",
    importance=5
)

# 获取复习队列
queue = cms.get_review_queue()

# 记录复习
cms.record_review(note_id, rating=4)

# 模拟交易决策
result = cms.simulate_decision({
    "market_trend": "TREND_UP",
    "buy_score": 75,
    "time_window": "T2"
})

# 生成报告
report = cms.generate_daily_report()
```
