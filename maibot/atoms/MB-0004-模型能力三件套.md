# 模型能力三件套

## 原子结论
MaiBot 的模型配置不是单个模型名，而是按能力使用不同模型：LLM 负责文本推理与回复，VLM 负责图像理解，Embedding 负责记忆检索与语义相似度。

## 关键事实
- 最小可用配置需要 LLM。
- 看图片需要 VLM。
- 记忆和语义检索能力依赖 Embedding。
- API Key、base_url、模型名和提供商余额是模型排错的高频点。

## 排错入口
遇到不回复、模型调用失败、401/402/403、连接超时，优先检查 `model_config.toml` 的 provider、key、模型名和网络。
