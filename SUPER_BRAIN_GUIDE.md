## 核心能力详解

### 1. 博弈论引擎 (Game Theory Engine)

**核心功能：**
- 识别5种市场博弈类型：零和博弈、协调博弈、斗鸡博弈、智猪博弈、囚徒困境
- 检测主力操盘6大套路：吸筹、拉升、洗盘、派发、假突破、逼空
- 反身性分析：识别自我强化的价格循环
- 逆向思维：站在对手角度预判行动
- 多步推演：预判对手下一步和下下一步

**典型场景：**
```
股价突然拉升 → 问自己：这是真突破还是诱多？
主力在做什么？ → 如果我卖出，主力会怎么反应？
```

### 2. 市场侦探 (Market Detective)

**核心功能：**
- 资金流向追踪：大单vs散户识别
- 庄家行为识别：吸筹/拉升/派发/洗盘
- 筹码分布分析：获利盘/套牢盘
- 量价异动检测：异常交易预警
- 龙虎榜追踪：机构席位分析

**典型场景：**
```
今日成交量是均量的3倍 → 分析：是谁在买？主力还是散户？
尾盘快速拉升 → 警惕：这是吸筹还是诱多？
```

### 3. 消息验证引擎 (News Verification)

**核心功能：**
- 消息真伪鉴别：官方来源 vs 传言 vs 谣言
- 时效性判断：消息是否最新、是否已过时
- 市场消化评估：是否已被price in
- 炒作识别：判断是否人为制造的消息
- 提前泄露检测：判断是否有人提前知道

**典型场景：**
```
突发利好新闻 → 先验证：这条新闻是真是假？
以前是否已经出现过类似新闻？
主力是否提前知道？
```

### 4. 情景推演引擎 (Scenario Planner)

**核心功能：**
- 多情景生成：牛市/熊市/震荡/突破/黑天鹅
- 概率评估：每个情景的主观概率
- 蒙特卡洛模拟：大规模随机模拟
- 决策树分析：多阶段最优路径
- 反脆弱策略：考虑极端情况和黑天鹅

**典型场景：**
```
面对不确定性 → 不是预测一条路，而是推演所有可能的情景
评估每个情景的概率和影响，制定相应策略
```

### 5. 反坐庄检测 (Anti Gaming)

**核心功能：**
- 识别吸筹信号：缩量下跌、支撑位防守、尾盘拉升
- 识别洗盘信号：快速下跌快速反弹、假破位
- 识别出货信号：高位放量滞涨、利好掩护
- 识别假突破：量能不足、快速回落、无跟风
- 生存分数评估：散户在当前环境的生存能力

**典型场景：**
```
股价突破后快速回落 → 这是假突破陷阱！
追高买入必被套
```

### 6. 超级认知引擎 (Super Cognitive)

**整合以上所有能力，实现：**
- 快速决策 vs 慢速思考：适时切换模式
- 元认知监控：思考自己的思考，识别盲点
- 知识融合：跨领域知识的创新组合
- 闭环学习：决策→执行→复盘→改进

## 思考流程示例

### 遇到突发新闻时的思考流程：

```
1. 【接收信息】
   昆仑万维：突发AI芯片利好，股价快速拉升3%

2. 【消息验证】(news_verification)
   - 验证真伪：来源可信度85%
   - 时效性：最新发布
   - 是否炒作：嫌疑度40%（中等）
   - 历史对比：3个月前有过类似消息，之后涨了5%

3. 【资金分析】(market_detective)
   - 主力参与度：65%
   - 资金流向：净流入
   - 筹码分布：获利盘75%

4. 【博弈分析】(game_theory)
   - 博弈类型：零和博弈
   - 主力意图：可能在拉升出货
   - 反身性：已形成正反馈循环

5. 【情景推演】(scenario_planner)
   - 情景1（概率40%）：继续拉升 → 持有
   - 情景2（概率35%）：冲高回落 → 止盈
   - 情景3（概率15%）：横盘整理 → 观望
   - 情景4（概率10%）：黑天鹅 → 止损

6. 【反坐庄检测】(anti_gaming)
   - 检测结果：可能是拉升阶段
   - 生存分数：65/100
   - 风险等级：中

7. 【综合判断】(super_cognitive)
   - 置信度：70%
   - 推荐行动：顺势持有，设定止盈位
   - 备选方案：不追高，等回调再买
```

## 设计理念

### 为什么这样做？

传统AI的局限：
- 只看表面信息，不深挖背后逻辑
- 缺乏逆向思维，不会站在对手角度
- 不会验证信息真伪
- 单一路径预测，不考虑多种可能

超级博弈大脑的优势：
- 像顶级交易员一样思考
- 多维度交叉验证
- 博弈论视角看市场
- 多情景推演备选方案
- 持续从错误中学习

### 核心原则

1. **风险第一**：永远先考虑风险，再考虑收益
2. **逆向思维**：站在对手角度想问题
3. **多情景推演**：不做单一预测，考虑所有可能
4. **闭环学习**：每个决策都要复盘总结
5. **极度透明**：不隐瞒不确定性，承认自己的盲点
