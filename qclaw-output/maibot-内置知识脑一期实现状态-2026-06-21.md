# MaiBot 内置知识脑一期实现状态

日期：2026-06-21

## 已完成

- MaiBot 新增内置知识脑服务层 `src/maisaka/brain/`
- 新增内置工具 `brain`，支持：
  - `search`
  - `consult`
  - `digest`
  - `update`
  - `read`
  - `export`
  - `task_add`
  - `task_list`
  - `task_update`
- 工具执行后的高价值结果，优先回灌到内置知识脑，再降级到原有 HTTP 第二大脑
- 内置知识脑现在可在缺少 A_Memorix 依赖时独立启动，避免被外部长期记忆模块绑死
- 搜索增加了短语命中和本地缓存，适合后续逐步扩大数据量
- Planner 组装上下文时会自动注入内置知识脑参考
- reply 成功后会把重要对话沉淀到内置知识脑
- reply effect 结案后会把复盘经验沉淀到内置知识脑
- 中期聊天摘要会自动沉淀到内置知识脑，形成“对话主题级”长期经验
- reply 在生成前会先向内置知识脑做一次轻量自检检索，作为回复理由参考
- 内置知识脑的输出已经改成更像“记忆卡片”的格式，便于人类和模型一起阅读
- 内置知识脑已加入 `palace / room / shelf` 三层空间定位，可把记忆组织成“宫殿 / 房间 / 货架 / 卡片”
- 搜索和决策咨询现在会先推断当前主题所在的心智空间，再优先从对应宫殿召回记忆
- `brain` 工具已经支持显式传入 `palace` / `room` / `shelf`，便于 WorkBuddy、QClaw、人工整理统一写入
