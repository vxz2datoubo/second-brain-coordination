## 二、各层详细设计

### Layer 1: 入市信号层 (Entry Signals)

**理论基础**：海龟法则的20日突破理论 + 华泰A股情绪分位策略 + KDD2024多模态融合

**信号来源**（4维共振）：

| 维度 | 指标 | 权重 | 买入触发条件 |
|------|------|------|------------|
| 量价 | Volume Profile支撑带触及 | 30% | 价格进入S1-S2支撑带 |
| 资金 | 主力净流入方向 | 30% | 昨日/当日主力净流入>0 |
| 情绪 | 5层情绪评分 | 20% | 情绪< -1.0 (偏恐惧/极度恐惧) |
| 板块 | 关联股同步性 | 20% | 板块同涨或个股独立强势 |

**买入评分卡**：
```
量价进入支撑带    +3分
主力资金净流入    +3分  ← 最重要
情绪偏恐惧        +2分
板块共振确认      +2分
───────────────────
总分 ≥ 5分 → 买入信号 (绿灯)
总分 3~4分 → 观望信号 (黄灯)
总分 < 3分 → 禁止买入 (红灯)
```

**质疑检验**：
- ❓ 4维共振会不会太严格，错失机会？→ 设黄灯区允许半仓试探
- ❓ 资金流向数据T+1滞后怎么办？→ 用盘口大单成交辅助判断当日方向

### Layer 2: 仓位管理层 (Position Sizing)

**理论基础**：凯利公式(Kelly Criterion) + ATR波动率仓位 + 海龟N值

**核心公式**：
```
目标仓位 = min(总资金 × 5%, 总资金 / (止损幅度 × 100))

示例：总资金100K
  昆仑止损0.5元 → 最大仓位 = 100K/(0.5×100) = 2000股 ≈ 8万
  蓝标止损0.2元 → 最大仓位 = 100K/(0.2×100) = 5000股 ≈ 6.8万
  
单票上限：总资金 × 10%
当天T仓上限：总资金 × 5% (波仔现有规则)
```

**仓位分档**：

| 档位 | 触发条件 | T仓比例 | 底仓调整 |
|------|---------|---------|---------|
| 满仓 | 情绪<-2.0 + 支撑带 + 主力流入 | 5% | 不动 |
| 正常 | 情绪-1~+1 | 5% | 不动 |
| 减仓 | 情绪>+2.0 贪婪 | 3% | 卖20%底仓 |
| 空仓 | 情绪>+2.5 极度贪婪 | 0% | 卖50%底仓 |

**质疑检验**：
- ❓ 凯利公式需要胜率和赔率作为输入，我们的胜率数据够吗？→ 先用手动统计，50笔后自动计算
- ❓ 止损幅度用固定金额还是ATR动态？→ 先用固定（方便执行），后续升级ATR

### Layer 3: 止损风控层 (Risk Management)

**理论基础**：海龟2N止损 + 塔勒布黑天鹅理论 + 行为金融损失厌恶纠正

**三重止损**：

| 类型 | 规则 | 触发条件 |
|------|------|---------|
| **硬止损** | 单笔亏损 > -2% | 无条件割，不容讨论 |
| **时间止损** | 持有 > 30分钟不盈利 | 平推出局 |
| **趋势止损** | 连续2根5分钟K破支撑 | 趋势已坏，立即离场 |

**每日熔断**：
```
单日累计亏损 > 总资金 × 2% → 全天强制停止交易
单日连续亏损3笔      → 全天强制停止交易
单日亏损后追单1次    → 禁止再交易（防追高症）
```

**今天下午波仔的教训**：
```
T3卖40.70→T4追40.74→T5追40.43→利润归零
如果当时有每日熔断：T4平推出局后触发禁手令
```

**质疑检验**：
- ❓ -2%硬止损会不会在盘中V反时太严格？→ 回顾：蓝标昨天从13.64跌到13.01(-4.6%)，-2%止损能救3%
- ❓ 有没有例外情况可以放宽？→ 无例外。硬规则不需要人工干预

### Layer 4: 出场信号层 (Exit Strategy)

**理论基础**：波仔层T回笼法 + Volume Profile压力带

**两种出场场景**：

| 场景 | 触发 | 执行 |
|------|------|------|
| **止盈出场** | 价格触及P1压力带 | 卖T仓，记录差价 |
| **接回出场** | 价格回落到卖出价以下 | 按波仔层T回笼法，优先接亏最大的 |
| **时间出场** | 14:30仍未成交 | 强制平仓，不隔夜 |

**止盈锚点**（基于Volume Profile）：
```
卖出后，压力带P1 = 第一止盈区（概率70%）
        压力带P2 = 第二止盈区（概率30%）
        压力带P3 = 不设止盈目标（概率<10%，对T仓不现实）
```

**质疑检验**：
- ❓ 到了P1要不要等P2？→ 历史回测：P1→P2胜率仅30%，不值得赌
- ❓ 冲过P1怎么办？→ 底仓享受溢价，T仓已锁利不后悔

### Layer 5: 战术执行层 (Execution)

**理论基础**：算法交易TWAP/VWAP执行优化 + 竞价分两段机制

**三大执行窗口**：

| 窗口 | 时间 | 策略 | 风险 |
|------|------|------|------|
| 竞价窗口 | 9:15-9:25 | 竞价裁判系统评分 | 9:15-9:20可撤→虚单识别 |
| 开盘窗口 | 9:30-9:35 | 方向确认后挂单 | 不抢闪电战 |
| 盘中窗口 | 9:35-14:30 | Volume Profile锚点挂限价单 | 避免追高 |

**执行规则**：
- 永远用限价单，不用市价单
- 挂单价 = 支撑/压力锚点 ± 1个tick
- 卖后禁手30分钟（波仔铁律）

**质疑检验**：
- ❓ 限价单不成交怎么办？→ 等待优于追高。没买到比买贵了好50倍

---

### Layer 6: 复盘迭代层 (Review & Iterate)

**日度复盘清单**：

| 项目 | 内容 | 频率 |
|------|------|------|
| T仓战绩 | 每笔入场+出场+差价+胜负 | 每日 |
| 信号回顾 | 买入/卖出信号是否准确 | 每日 |
| 情绪追踪 | 5层情绪评分与实际走势对比 | 每日 |
| 变量分析 | 不可控变量识别（防过拟合）| 每周 |
| 策略优化 | 基于≥50笔样本做参数微调 | 每50笔 |

**迭代原则**（防过拟合）：
```
1. 最少50笔样本才能改参数
2. 每次只改1个变量
3. 改完回测验证，不改了就继续跑
4. 同变量同样结果反复出现→形成规则
5. 不同变量不同结果→不是规则，不加入
```

### Layer 7: 心理纪律层 (Trading Psychology)

**理论基础**：行为金融四大偏差 + 波仔实测性格

| 偏差 | 波仔表现 | 纠正机制 |
|------|---------|---------|
| **损失厌恶** | 蓝标跌-4%不肯割 | -2%硬止损，无例外 |
| **过度自信** | 赚了后连续追高 | 日盈3笔后禁手 |
| **锚定效应** | 死盯成本价13.64 | 只看支撑/压力，忘记成本 |
| **处置效应** | 赚的急着走，亏的死扛 | 止盈止损都执行，不等 |

**迪迪的角色**：
- 强硬督促止损（不能软）
- 制止追高（禁手30分钟铁律）
- 在极度恐慌时鼓励入场
- 在极度贪婪时强制减仓

**质疑检验**：
- ❓ AI督促纪律真的有效吗？→ 今天上午昆仑两次卖点做到了，有进步
