## 一、仓位铁律

```
总仓位 = 底仓(97%) + T仓(3%)

底仓 → 长线不动，等AI板块反弹周期
T仓  → 每日最多2笔，降底仓成本
```

**原则：T仓盈亏对总资产影响微乎其微（3%×2%=0.06%），做对了捡钱，做错了就当没发生。不值得为T仓盈亏焦虑。**

## 二、每日操作流程

### 09:30 开盘
- 看开盘方向（高开/低开）
- 根据历史规律决定今日策略：

| 开盘类型 | 昆仑万维 | 蓝色光标 |
|----------|----------|----------|
| 高开>0.5% | 优先正T（不做倒T） | 倒T优先 |
| 低开<-0.5% | **倒T优先** | 倒T优先 |
| 平开 | 观望 | 观望 |

### 09:35 第一波信号
- 急跌不要追着卖，等反弹
- 急涨不要追着买，等回调

### 10:00 关键窗口
- 上午V反通常在这个时间出现
- 看到V反到均价上方 → **立即卖出第一笔（不许多想）**
- 如果拉到日内高点附近 → 卖第二笔

### 14:00 决断点
- 还没出信号 → 这一天大概率就这样了
- 有持仓未出 → 可以等尾盘但别抱太大希望

### 14:50 最后底线
- 剩余仓位全部清掉

## 四、核心教训

### 2026-06-26 实盘复盘

```
昆仑万维: 低开→砸38.83→V反40.33→回落38.73收盘
蓝色光标: 低开→砸13.42→V反13.94→回落13.45收盘

最佳操作:
  ❌ 09:35 急跌时追着卖 → 不该卖（地板价）
  ✅ 10:24 反弹到40.33 → 该卖（全天唯一卖点）
  ❌ 之后的3小时 → 全是垃圾时间，没有任何机会
```

### 记住三点

1. **卖点稍纵即逝** — 今天10:24只有一个窗口，错过了就没了
2. **下午大多是垃圾时间** — 上午不拉，下午更不会拉
3. **3%仓位不值得焦虑** — 亏2%对总资产影响0.06%，别被情绪绑架

## 五、买卖点速查

### 倒T（先卖后买）

| 条件 | 阈值 |
|------|------|
| 卖出信号 | 价格 > VWAP + 0.5%，且RSI > 50 |
| 第1笔卖出 | 回到均价上方立即卖 |
| 第2笔卖出 | 拉到日内高点附近再卖（拉开价差） |
| 接回目标 | 卖出价或以下 |
| 卖飞处理 | 涨超卖出价3% → 等高位支撑确认后追回，追回后冷却20分钟不新卖 |

### 正T（先买后卖）

| 条件 | 阈值 |
|------|------|
| 买入信号 | 跌到支撑位 + 止跌放量 |
| 卖出目标 | 保本出或微利（1-3%） |
