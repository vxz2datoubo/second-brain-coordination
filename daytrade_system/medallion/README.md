# 大奖章T仓交易系统

基于你的实战经验打造的智能T仓辅助系统。

## 核心功能

### 1. 实时信号生成
- 开盘紧急T检测（0.5%波动机会）
- 冲高滞涨信号（1-3%+放量+滞涨）
- 分时均线偏离分析
- 情绪过滤（利好/弱势自动调整）

### 2. Web仪表盘
访问 http://localhost:8767

功能：
- 两只股票实时信号展示
- 情绪切换（整体强势/弱势/正常）
- 手动记录买卖操作
- 每日T仓状态追踪

### 3. 情绪监测
- AI/游戏板块新闻关键词监测
- 基于情绪自动调整信号权重
- 突发消息提醒

## 信号说明

### 置信度等级
- A++: 强烈信号，强烈建议执行
- A: 良好信号，建议执行
- B: 中等信号，可执行
- C: 偏弱，谨慎执行
- D: 信号弱，观望

### 状态说明
- **建议卖出**: 多个指标共振，出现卖出机会
- **建议接回**: 价格回到均线/卖出价附近
- **持仓观察**: 有持仓，等待机会
- **观望**: 信号不强，等待更好机会

## 交易规则

根据你的实战经验：

1. **开盘紧急T**: 开盘15分钟内涨跌超0.5%，快速高抛低吸
2. **冲高滞涨卖**: 累计涨1-3%+成交放大+开始滞涨
3. **均线接回**: 价格回落到分时均线附近买回
4. **情绪过滤**: 利好消息+板块强势时不追卖

## 回测发现

从449天历史数据中发现的规律：

### 有效的
- 自然接回策略确实有效（信号能找到相对高点）
- 胜率可达80%+

### 需注意
- 趋势股卖出后经常继续涨
- 日末强平是最大亏损来源
- 需要结合情绪判断
