# 美学偏好学习系统 - 设计文档
版本: v1.0 | 日期: 2026-07-07

---

## 核心理念

系统能够通过用户提供的图片，自动分析并学习用户的审美偏好，
发现用户自己都没有意识到的隐藏特征。

---

## 系统架构

```
用户发图
    ↓
全维度视觉分析 (LLM Vision)
    ↓
存入知识库 (美学偏好库)
    ↓
多图积累后 → 统计偏好频率
    ↓
发现隐藏模式 (用户自己没意识到的)
    ↓
可视化展示偏好图谱
    ↓ (可选)
图像生成 (基于偏好生成新图)
```

---

## 分析维度

### 1. 人物特征
- 面部: 脸型、眼睛、眉毛、鼻子、嘴唇、下巴
- 发型: 长发/短发/卷发/直发、发色、刘海
- 表情: 微笑/无表情/大笑、眼神、气质
- 年龄感: 青春/成熟/少女感/御姐
- 身材: 高挑/娇小/丰满/苗条/匀称

### 2. 服装风格
- 整体风格: 韩系/日系/欧美/休闲/正式/运动/文艺
- 颜色: 莫兰迪/高饱和/黑白/彩色系
- 上装: T恤/衬衫/吊带/卫衣/毛衣/外套
- 下装: 牛仔裤/短裙/长裙/短裤/阔腿裤
- 材质: 蕾丝/针织/棉麻/丝绸/皮革

### 3. 场景氛围
- 地点: 室内/海边/城市/自然/咖啡厅/街道/天台
- 光线: 自然光/逆光/柔光/暗调/明亮
- 时间: 白天/黄昏/夜晚/清晨
- 色调: 暖色调/冷色调/中性
- 滤镜感: 日系/韩系/胶片/高饱和/低饱和

### 4. 构图细节
- 景别: 特写/近景/中景/远景/全身
- 角度: 正面/侧面/俯视/仰视
- 背景: 虚化/清晰/纯色/杂乱
- 构图: 三分法/中心/对称/留白

### 5. 互动元素
- 配饰: 帽子/眼镜/项链/耳环/手表
- 季节: 春/夏/秋/冬
- 物品: 花/书/宠物/咖啡
- 互动: 抱宠物/拿道具/与场景互动

## 知识库结构

```
wisdom-brain/knowledge/aesthetics/
├── _index.json              # 索引和统计
├── person/                  # 人物偏好
│   ├── face.json           # 脸型偏好
│   ├── eyes.json           # 眼睛偏好
│   ├── hair.json           # 发型偏好
│   ├── expression.json     # 表情偏好
│   └── body.json           # 身材偏好
├── clothing/               # 服装偏好
│   ├── style.json         # 风格偏好
│   ├── color.json         # 颜色偏好
│   ├── top.json          # 上装偏好
│   └── bottom.json       # 下装偏好
├── scene/                 # 场景偏好
│   ├── location.json     # 地点偏好
│   ├── mood.json        # 氛围偏好
│   └── light.json       # 光线偏好
├── composition/           # 构图偏好
│   ├── angle.json       # 角度偏好
│   └── framing.json     # 景别偏好
└── hidden/              # 隐藏模式 (用户没意识到的)
    └── patterns.json   # 发现的隐藏模式
```

## 核心功能

### 1. 图片分析
- 接收图片路径
- LLM全维度分析
- 返回结构化特征

### 2. 偏好统计
- 记录每张图的特征
- 统计各特征出现频率
- 排序高频偏好

### 3. 隐藏模式发现
- 分析共现关系
- 发现规律组合
- 挖掘用户无意识的偏好

### 4. 偏好报告生成
- 生成文字描述
- 生成可视化图谱
- 生成图像生成提示词

---

## API接口

### POST /api/aesthetics/analyze
分析单张图片

### GET /api/aesthetics/summary
获取偏好总结

### GET /api/aesthetics/hidden
获取隐藏模式

### POST /api/aesthetics/generate
基于偏好生成图像 (需API)
