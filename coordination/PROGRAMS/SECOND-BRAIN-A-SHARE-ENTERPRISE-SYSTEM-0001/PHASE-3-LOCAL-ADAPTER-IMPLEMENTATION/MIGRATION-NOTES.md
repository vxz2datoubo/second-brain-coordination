# Migration Notes

The synthetic adapter does not migrate or read `F:\aidanao` runtime state. A later approved source-binding task must provide a source manifest, isolate a copy of one approved sample, map field semantics, declare point-in-time availability and A-share policies, then run the same contract suite plus source-specific tests.

Rollback is a branch/PR revert. No database, JSONL, bulletin, service registry, local source file, or trading state is changed by this phase.
