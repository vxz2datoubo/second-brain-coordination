# 用户指令：凭证类秘密仅本地保存

状态：ACTIVE  
日期：2026-07-21

## 规则

- 用户授权知识库完整学习、保存和检索全部普通知识内容。
- 访问令牌、接口密钥、登录密码、会话凭证、私钥及同类认证秘密，不得上传到GitHub、公共对象存储或普通云端知识索引。
- 这些凭证类秘密只保留在用户本地受控存储中，并保持本地可检索和可定位。
- 知识图谱、LearningPacket和ContextBundle仅记录引用ID、用途、所属系统、更新时间、校验摘要与本地位置说明，不记录实际秘密值。
- API文档、接口说明、字段定义和非认证技术资料不属于凭证类秘密，应正常完整入库。
- 本规则只限制凭证值的存储位置，不限制其他知识的完整学习、融合、检索和GPT语义访问。
