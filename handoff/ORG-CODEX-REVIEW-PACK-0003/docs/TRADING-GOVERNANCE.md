## 一、操作流程

### 实时监控流程（每次查看昆仑 300418 时）

1. 查看板块排行（TOP5 + BOTTOM5）→ `data_sector ranking`
2. 检查涨跌超过 5% 的板块 → 触发虹吸检测
3. 读取最新 5 条个股新闻 → `data_news`
4. 检查大盘涨跌（上证指数 + 沪深 300）
5. 板块异动 > 5% → **立即预警**，不等用户问
6. **"钱去哪了"优先级 > "DDX 怎么样"**

### 全面分析流程（用户说"全面分析/全系统分析"时）

触发以下 15 项技能联动：
1. Phase 结构判断
2. 四渠道资金（特大单/大单/中单/小单）
3. 供应测试
4. 资金池 FIFO
5. 锚定盈亏
6. 蓝标 300058 对比
7. 旋转门潮汐
8. 因果法则目标推演
9. 量价剖面
10. 板块雷达（避险 vs 调仓）
11. 恐慌指数
12. 大盘关联度
13. DDX 方向
14. 盘口
15. 全局盘面感知（含仓位建议）

> 注：此为工作流预设，非全局铁律。是否需要始终调用全部 15 项由当前任务上下文决定。

### 集合竞价分析流程

1. 阶段一（9:15-9:20，可撤单）→ 观察是否有"烟雾弹"（大单挂涨停后撤掉）
2. 阶段二（9:20-9:25，不可撤单）→ 判断匹配价方向和量能
3. 阶段三（9:25 开盘价）→ 结合昨日收盘、竞价量判断当日方向
4. 与 Wyckoff Phase 对应：高开 + 放量竞价 + 前日缩量十字星 = 最强看涨

## 四、高价值预警系统

### 运行方式
- 被动查询：`python core/alert_engine.py query 7`
- 主动推送：每 2h 自动化系统级扫描
- 数据源：WeStock `data_hot`（全市场热新闻）+ `tool_event`（42类事件）

### 评估维度
- impact（热度）× urgency（时效）× relevance（关联）
- 阈值：55（`alerts/rules.json`）
- 关键词甄别：预告类（将于/预告/即将/下周）、重大类（突发/暴增/暴跌/发射/制裁）
