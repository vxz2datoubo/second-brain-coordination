## 二、资金身份推断假设

### H03: 特大单/大单 = 纯机构（散户无法伪装）
- **原文**: "特大单/大单=纯机构(散户无法伪装)"
- **状态**: ⚠️ 部分验证 — 小额订单可能为散户，大额订单大概率是机构；但量化基金的拆单可能落在中单/小单区间
- **冲突**: WeStock 与 Tushare 的大单阈值定义可能不同
- **建议**: 保留为弱假设，不用于强推断

### H04: 大小单同向 = 3.8x 增强
- **原文**: "共识>分歧: 大小单同向=3.8x增强"（来源：a-share-call-auction 技能文档）
- **状态**: 🔬 待验证 — 数字 3.8 的来源不明确，无独立验证脚本
- **需验证**: 大小单同向 vs 反向的后续收益差异，计算实际增强倍数
- **建议**: 标记为待验证，不可在策略中硬编码

### H05: OHLCV 代理无效 (MF r=0.294)
- **原文**: "OHLCV代理无效(MF r=0.294)"
- **状态**: 🔬 待验证 — 有 r 值声明但无完整的复现脚本
- **需验证**: 对 300418 和 300058 重新计算 Money Flow index 与真实资金流的相关系数
