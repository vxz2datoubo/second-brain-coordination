# FIELD-SEMANTICS-REVIEW — 字段语义独立复验

> 原则: 不沿用旧推测。每个字段给出原始值/时序变化/类型/适用标的/数值关系/官方定义状态/最窄可确认语义。找不到官方定义 → `unknown_vendor_semantics`。

## 字段表（来源: TDX MCP ProInfo，R3/R4/R5 实测）

### InOut
- 原始值: 300418 R3=47,497,568 / R4=44,486,720 / R5=199,865,024；600519 R3=5,073,480 / R4=40,323,552；510300 R3=20,091,696 / R4=50,850,368
- 类型: float（带小数，如 000609 R3=-1,693,695.5）
- 适用: 全部6标的（含指数 999999 R3=2,433,148,930）
- 时序变化: 跨轮大幅变动，可正可负（000609 R3负 / 300418 正）
- 数值关系: 量级与 Volume/Amount 同级（委买委卖量级），非价格
- 官方定义: **未找到**（无 tqcenter 字段注释，无 MCP 文档）
- 最窄可确认语义: **某委买/委卖净额聚合量**（单位疑为股/手，正负方向未知确切定义）
- 不允许推断: 不能确定是"总委买−总委卖"还是"净委买量"还是"委买-委卖差额"；旧报告"总委买−总委卖量净值"为**推测，未经证实**
- 状态: `unknown_vendor_semantics`

### InOutHB
- 原始值: 300418 R3=-9,637,880 / R4=-29,447,936 / R5=+65,010,496（**正负翻转**）；510300 R3=-108,894,784 / R4=-377,355,072（巨幅）；600519 R3=22,703,400 / R4=22,782,528
- 类型: float
- 适用: 全部6标的
- 时序变化: 跨轮剧变，510300 量级远超 InOut
- 数值关系: 与 InOut 同量级但独立变动，无固定比例
- 官方定义: **未找到**
- 最窄可确认语义: **某 L2 委托类聚合量，实时变动，可正可负**
- 不允许推断: **旧报告"撤买/撤卖聚合"为推测，无官方文档/可重复实验支持 → 推翻，标 unknown**
- 状态: `unknown_vendor_semantics`（**纠正旧报告的语义断言**）

### Wtb
- 原始值: 300418 R3=27.70 / R4=14.17 / R5=0.39；300058 R3=39.77 / R4=-23.57（翻负）；510300 R3=-17.24 / R4=53.14
- 类型: float（百分比量级 -60~+60）
- 适用: 全部标的（指数无 Wtb）
- 数值关系: 量级像"委比%"，与 InOut 正负方向**不完全一致**（510300 R3: InOut正/Wtb负）
- 官方定义: 未找到；TDX 公式中"委比"=委买-委卖/委买+委卖，Wtb 量级吻合
- 最窄可确认语义: **疑为委比%，但与 InOut 正负不一致需进一步验证**
- 状态: `unknown_vendor_semantics`（高度疑似委比，但未证实）

### NowVol
- 原始值: 300418 R3=1665 / R4=222 / R5=227；600519 R3=49 / R4=59
- 类型: int (string)
- 官方定义: 未找到
- 最窄语义: **某最新量指标**（R3→R4 大降，疑为"最新成交量"或"现量"）
- 状态: `unknown_vendor_semantics`

### TotalBuyv / TotalSellv（仅指数 999999）
- 原始值: TotalBuyv R3=631 / R4=430；TotalSellv R3=1569 / R4=1819
- 类型: int (string)
- 适用: **仅上证指数**（个股无此字段）
- 数值关系: Sellv>Buyv（卖压），与指数下跌方向一致（上证 R3 -0.95%→R4 -1.07%）
- 官方定义: 未找到
- 最窄语义: **指数级委买/委卖总量**（高度疑似，与涨跌方向一致）
- 状态: `unknown_vendor_semantics`（疑似委买卖总量，未证实）

### CasImbVol / CasUpperPx（仅指数 999999）
- 原始值: CasImbVol R3=2.19863729e-42 / R4=2.54896191e-42；CasUpperPx R3=8.84e-43
- 类型: float（**极小浮点 ~1e-42，非物理量纲**）
- 官方定义: 未找到
- 最窄语义: **UNKNOWN**（量级异常，疑为未初始化/编码错误/占位，不可作业务字段）
- 状态: `unknown_vendor_semantics` + **疑似脏数据，禁用**

### HQMaskFlag
- 原始值: 300418=11 / 300058=10 / 600519=7 / 510300=6 / 000609=1048576(0x100000)
- 类型: int (string)
- 官方定义: 未找到
- 最窄语义: **L2 能力位掩码**（不同标的不同，*ST 为 0x100000）
- 状态: `unknown_vendor_semantics`

## 结论
- **所有 ProInfo L2 字段均无官方定义佐证**，旧报告对 InOut/InOutHB 的具体语义断言（"总委买−总委卖"/"撤单聚合"）**均属推测，本轮推翻，统一标 unknown_vendor_semantics**。
- 仅可确认: 这些字段实时变动(RAW_EVIDENCE_VERIFIED)、为聚合量(AGGREGATE_ONLY)、与涨跌方向有弱相关(TotalBuyv/Sellv)。
- **决策红线**: 在取得 TDX 公式编辑器官方定义或可重复对照实验前，**不得用 InOut/InOutHB 做方向性资金流判断**。
