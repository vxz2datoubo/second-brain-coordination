1. **.th2 / .tcu / .tnf / PriCS.dat 为 TDX 专有压缩格式**，可能含 L2 十档/快照数据（.th2 14MB 量级与十档行情缓存吻合；PriCS.dat 802KB 疑=优先快照）。
2. 这些文件**会话开始(09:21:50-58)写入**；其中 **sz.tfz 持续实时更新(09:42)**，其余 .th2/.tcu/.tnf mtime 冻结在 09:21:58（可能为登录时全量加载，运行时内存更新不回写磁盘）。
3. **未发现 .l2d / .tick**（旧报告此点正确）。
4. **不能解析**: 均为二进制专有格式，需 TDX 官方库或逆向。本次未做哈希差分（无"开通L2前"基线快照）。
