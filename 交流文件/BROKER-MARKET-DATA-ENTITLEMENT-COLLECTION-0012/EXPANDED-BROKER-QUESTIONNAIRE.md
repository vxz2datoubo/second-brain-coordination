## B. 行情颗粒度

请逐项确认(标注✅支持/❌不支持/⚠️有条件):

- [ ] 五档快照
- [ ] **十档快照** (重点)
- [ ] **逐笔成交** (raw trade tick, 非3秒聚合) (重点)
- [ ] **逐笔委托** (raw order/entrust event) (重点)
- [ ] **买一卖一委托队列** (order queue depth) (重点)
- [ ] **千档委买委卖队列** (如迅投xtquant的l2thousand_queue)
- [ ] 总委买量 / 总委卖量
- [ ] 总撤买量 / 总撤卖量
- [ ] 单笔撤单事件
- [ ] **集合竞价虚拟参考价** (09:15-09:25)
- [ ] **匹配量** (09:20-09:25)
- [ ] **未匹配量**
- [ ] **未匹配方向**
- [ ] **09:25 开盘撮合记录** (集中撮合明细)
- [ ] **14:57-15:00 深市收盘集合竞价**轨迹
- [ ] **历史Level-2回放** (历史tick/逐笔/委托可重放)

**特别澄清**: 如答"支持Level-2", 请明确是指仅十档快照+聚合统计, 还是包含逐笔成交+逐笔委托+委托队列。我们需要区分 "L2_AGGREGATE" 和 "RAW_TRADE_TICK"。
